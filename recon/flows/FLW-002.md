## Step sequence (happy path — "user correctly diverts to police")

```
STEP 1 — trigger_point
  ACTION: navigate to /formular
  KIND:   trigger_point.

STEP 2 — data_collection_point
  ACTION: locate the "Kdy volat Policii ČR?" card heading
  DETAIL: const card = page.getByRole('heading', { name: /Kdy volat Policii ČR/i });
  KIND:   data_collection_point.

STEP 3 — trigger_point
  ACTION: click the card to expand the police-call panel
  DETAIL: await card.click();
  KIND:   trigger_point.

STEP 4 — assertion
  ACTION: assert all 7 hard-stop conditions are listed
  DETAIL: const items = page.locator('role=listitem')
              .filter({ hasText: /(zraněný|200 000|3\. osoby|prospěšné|plynulost|ujel|zvěř)/ });
          await expect(items).toHaveCount(7);
  KIND:   assertion — content+count check on the regulatory list.

STEP 5 — control_point
  ACTION: branch on test variant — happy ("call police") vs failure
          ("self-record despite condition")
  KIND:   control_point.

STEP 6 — assertion (happy variant)
  ACTION: assert the "Volat Policii ČR (158)" link is present and
          href is exactly `tel: 158` (note the literal space — observed)
  DETAIL: const policeLink = page.getByRole('link', { name: /Volat Policii ČR/i });
          await expect(policeLink).toBeVisible();
          await expect(policeLink).toHaveAttribute('href', /^tel:\s*158$/);
  KIND:   assertion — verifies the regulatory escape hatch works.

STEP 7 — data_collection_point
  ACTION: capture URL — must remain on /formular (NOT advanced to
          wizard step 1)
  DETAIL: const url = page.url();
  KIND:   data_collection_point.

STEP 8 — assertion
  ACTION: assert URL matches /formular(?:\?.*)?$
  KIND:   assertion — confirms the user was not advanced.
```

## Failure mode (mandatory per R-FAIL-1)

```
FAILURE F1 — wizard advances despite police condition
  TRIGGER: regression where the panel-internal "Vyplnit záznam o nehodě"
           button incorrectly advances the user past the gateway
  EXPECTED: test fails (this is a regulatory-grade hazard);
            evidence: URL after click + screenshot
  KIND mapping: trigger_point (click panel-internal button) +
                assertion (URL must NOT change to a wizard step path)

FAILURE F2 — tel:158 link missing or misformatted
  TRIGGER: the escape hatch is regressed (link removed or href changed)
  EXPECTED: test fails
  KIND mapping: data_collection_point + assertion
```

verification in a real iOS context — flagged in F2).
