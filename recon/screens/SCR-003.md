## Purpose

CS — Stránka s často kladenými otázkami; obsah je rozdělen do 6 kategorií
(Základní info / Technické požadavky / Vyplňování / Bezpečnost / Výstupy
/ Ostatní), každá s akordeonovými položkami. Volně přístupná
nezaregistrovanému návštěvníkovi.

EN — FAQ page; content is partitioned into 6 categories with accordion
items per category. Anonymously accessible.

## Key UI elements

- `<header>` `<banner>` — same as landing (logo, hamburger, CTA "VYPLNIT
  ZÁZNAM" → `/formular?utm_source=landing`).
- `<button>` "Zpátky na domovskou stránku" linking to `/`.
- `<heading>` "Často kladené otázky" — page H1.
- `<group>` "Platform" with 6 category buttons:
  `<button>` "Základní info"
  `<button>` "Technické požadavky"
  `<button>` "Vyplňování"
  `<button>` "Bezpečnost"
  `<button>` "Výstupy"
  `<button>` "Ostatní"
- For each currently visible category, a `<heading>` (the category name)
  followed by a series of `<heading>` + `<button>` + `<region>` accordion
  items. The "Základní info" category was the visible one during recon
  (5 items captured: "Co je to Bouračka?" / "Musím si dopředu něco
  stahovat?" / "Je používání Bouračky zdarma?" / "Jak postupovat při
  zranění?" / "Uznávají Bouračku všechny pojišťovny?").

## Notes

The accordion items are `<button>` + `<region>` pairs — well-suited to
ARIA assertion (Cypress + Playwright both support `getByRole('region')`
+ `getByRole('button', { expanded: true|false })` natively).
