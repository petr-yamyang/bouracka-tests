## Purpose

CS — Marketingový rozcestník: představuje produkt, vysvětluje vhodnost
("kdy záznam vyplnit"), zobrazuje 5-krokový průběh, ukazuje bezpečnost +
GDPR, FAQ akordeon, mediální blok. Primární CTA na všech řezech vede
do `/formular`.

EN — Marketing landing: introduces the product, eligibility ("when to
fill the record"), 5-step explainer, security/GDPR card, FAQ accordion,
press-kit block. Primary CTA across all sections routes into `/formular`.

## Key UI elements

- `<header>` `<banner>` — Bouračka.cz logo (link to `/`) + "Open menu"
  hamburger button + nav links (#kdy / #jak / #bezpecnost / #dotazy /
  #pro-media) + CTA "VYPLNIT ZÁZNAM" linking to `/formular?utm_source=landing`.
- Hero section — `<h1>` "Bouračka?" + sub-heading "Záznam o dopravní
  nehodě teď vyplníte pohodlně online" + tagline "Pro nehody bez zranění
  a bez nutnosti volat policii" + duplicate CTA "VYPLNIT ZÁZNAM" + hero
  image "Ilustrace mobilní aplikace Bouracka".
- 3-card explainer — Doklady / Fotky / Odeslat (each with one-sentence
  description; no inputs).
- "V jaké situaci se záznam o nehodě vyplňuje" — eligibility list
  (Nehody bez zranění / Odhad škody do 200 000 Kč / Nedošlo ke škodě
  na majetku 3. osoby) + button "Zobrazit informace kdy volat policii"
  (opens decision panel — see SCR-002 because the panel is shared).
- "Jak to probíhá?" — video play button (preview thumbnail) +
  ordered list of 5 steps + manual link
  (`https://www.ckp.cz/images/clanky/cz/bouracka/bouracka-manual-uzivatele.pdf`).
- "Bezpečnost a soukromí" — 2-bullet card + button to open GDPR memo
  panel.
- "Často kladené otázky" — 5-item accordion (each is a `<button>` toggling
  a `<region>`) + "Zobrazit všechny dotazy (FAQ)" link to `/faq`.
- "Pro média" — card linking to `/tz-ckp-bouracka.pdf?v=1`.
- `<contentinfo>` footer — info@bouracka.cz / Instagram / YouTube /
  Tým za projektem (ČKP logo) / GDPR memo link / Cookies (SUPIN CDN) /
  copyright stamp 2026.
