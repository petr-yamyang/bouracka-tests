## Purpose

CS — Záhlaví je viditelné napříč všemi stránkami SUT; nese logo
"Bouračka.cz" (link na `/`), hamburger pro mobilní rozbalení nav, sadu
kotev (#kdy / #jak / #bezpecnost / #dotazy / #pro-media — pouze na
landing) a hlavní CTA "VYPLNIT ZÁZNAM" → `/formular?utm_source=landing`.
Mobilní layout používá hamburger; desktop layout zobrazuje nav-odkazy
horizontálně.

EN — Header is visible cross-page; carries logo (link to `/`), hamburger
for mobile nav expansion, anchor-link set (only on landing), and primary
CTA "VYPLNIT ZÁZNAM" → `/formular?utm_source=landing`. Mobile uses
hamburger; desktop shows nav-links horizontally.

## Key UI elements

- `<banner>` wrapper.
- `<a href="/">` containing logo `<img alt="Bouračka.cz">` (also acts
  as home-link).
- `<button aria-label="Open menu">` (mobile-trigger; visible at all
  widths but particularly pushed at narrow viewports).
- `<navigation>` (only on `/`) with 5 anchor-links: `#kdy` `#jak`
  `#bezpecnost` `#dotazy` `#pro-media`.
- `<a href="/formular?utm_source=landing">` wrapping the primary CTA
  `<button>` with text "VYPLNIT ZÁZNAM".
