## Purpose

CS — Vstupní bod do průvodce vyplněním záznamu o dopravní nehodě.
Obrazovka přímo nezahajuje vyplňování; nejprve uživateli připomene
tísňové volání 112, kontrolní otázku "Kdy volat Policii ČR?" (7 hard-stop
podmínek) a zobrazí kartu se základními pokyny "Co dělat při nehodě?".
Primární CTA "VYPLNIT ZÁZNAM" zahájí samotného průvodce; reCAPTCHA odznak
je viditelný v pravém dolním rohu.

EN — Entry point into the wizard. Does not start data capture
immediately — the user is first reminded of the 112 emergency call,
shown the "Kdy volat Policii ČR?" decision (7 hard-stop conditions), and
presented with the on-scene checklist. Primary CTA "VYPLNIT ZÁZNAM"
launches the wizard. reCAPTCHA badge bottom-right.

## Key UI elements

- `<header>` `<banner>` — link to `/formular` (logo) + "menu" button
  (hamburger; same layout as landing).
- `<heading>` "Stala se vám dopravní nehoda? To zvládnete." (empathic
  hero copy; no inputs).
- Card 1 "Potřebujete okamžitou pomoc?" — wraps `<a href="tel:112">`
  with a generic descriptor + "Tísňové volání 112" label.
- Card 2 "Kdy volat Policii ČR?" — collapsed by default; expands to a
  side-panel listing the 7 hard-stop conditions + button "Volat Policii
  ČR (158)" (`<a href="tel: 158">`) + button "Vyplnit záznam o nehodě"
  (closes panel, returns to gateway).
- Card 3 "Záznam o dopravní nehodě" — generic descriptor "Vhodné pro
  malé nehody bez zranění a škody do 200 000 Kč vzniklé mezi 2 účastníky
  s českými občanskými průkazy." + primary CTA `<button>` "VYPLNIT ZÁZNAM"
  (`type=button`; advances into the wizard's step 1).
- "CO DĚLAT PŘI NEHODĚ?" — on-scene 3-bullet checklist (warning lights /
  triangle / call 112).
- Footer link "Zpracování osobních údajů" → `/formular/personal-data`.
- reCAPTCHA presentation node bottom-right (likely Google reCAPTCHA v3
  given the badge-only UX).

## Validation rules

buttons + tel-uri links + the GDPR-link. Validation rules N/A *for this
screen*; the wizard's step 1 (post-CTA) is where input validation begins
and is not in this recon.

## Notes

The "Kdy volat Policii ČR?" interlock is regulatory-grade content —
mis-routing here is a real-world hazard. R-FAIL-1 obligation is
non-negotiable: the interlock decision branch must have ≥ 1 failure
scenario asserting that the user who selects ANY of the 7 conditions is
NOT advanced into the self-record wizard.
